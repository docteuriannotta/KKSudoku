# Knight & King Sudoku

PWA Sudoku avec contraintes d'échecs (Cavalier & Roi).

**Jouer :** https://docteuriannotta.github.io/knight-king-sudoku/

## Fonctionnalités
- 3 variantes : Knight ♞, King ♚, Les deux ⚔
- 3 modes : Classique, Survie, Chrono
- 5 niveaux de difficulté
- La Quête des Gardiens — 7 boss à vaincre
- Succès, XP, rangs, statistiques
- Installable sur Android / iOS / Desktop
